Strategy for growing the databank

Keep every new file in the same JSON structure: question, options A-D, correct_answer, unit_number or study_unit, unit_title, subunit_number or subunit, subunit_title. Uploading a new JSON in this screen adds it only to your current browser; replacing GitHub's index.html makes it permanent for everyone who knows the password.
