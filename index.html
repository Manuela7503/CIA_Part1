Data / Add more questions

Important: GitHub Pages is static. The browser cannot directly save into GitHub's questions.json.

Source of truth: questions.json. This HTML contains no question database. The app always loads the current GitHub questions.json when it opens.

After importing, the new questions exist only in this browser session until you download the updated questions.json and replace the file on GitHub.

To make changes permanent: upload this downloaded file to GitHub root and overwrite the old questions.json. Keep the filename exactly questions.json.

Unsaved session changes

You imported extra questions. They are loaded now, but they are not saved to GitHub. Click Download UPDATED questions.json, then replace questions.json in your GitHub repository.

Editing strategy

For corrections: edit questions.json directly on GitHub, commit, and refresh the app. For adding bulk questions: import JSON files here, download the updated questions.json, then replace the GitHub file. A pure static GitHub Pages app cannot write back to GitHub automatically without an unsafe token or a backend.
