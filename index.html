The main question bank is now a separate editable file: questions.json. Correct questions there, commit it to GitHub, and the app will load the updated bank. You can still import extra JSON files temporarily on this device.

Strategy for growing the databank

Keep every new file in the same JSON structure: question, options A-D, correct_answer, unit_number or study_unit, unit_title, subunit_number or subunit, subunit_title. For a permanent update, edit or replace questions.json in GitHub. Imports in this screen are temporary for this browser only unless you export and commit the merged JSON as questions.json.
